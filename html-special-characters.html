Topics: Front End Development: Special Characters: Sept. 29th 2013

HTML Special Characters are symbols that require a specific code in order to have the symbol show up on your website

Things I Learned

  • Used to make special characters and accented letters show up on your web pages
  • These special set of codes are called character entities
  • The set of codes are input into your HTML code
  • The browser will then display the corresponding symbol or character
  • Exclamation marks, dollar signs, and asterisks are examples of symbols that require a special charater code

Summary

Special Carater codes are required in your code in order to create symbols, punctuation, and signs. If the codes are not used then there will be a big problem with the web code and the browser will not be able to read your code.
